# Digital Notice Board

A MERN stack web application for managing campus announcements, timetables, placements, and events.

## Tech Stack
- React (Vite)
- Node.js
- Express
- MongoDB
- TailwindCSS

## Features
- Dashboard UI
- Category filtering
- Pinned notices
- Search functionality
- Admin notice management

## Run Locally

Backend
cd server
npm install
npm run dev

Frontend
cd client
npm install
npm run dev
